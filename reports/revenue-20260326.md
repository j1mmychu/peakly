## Critical Finding: Travelpayouts Marker Is a Placeholder

Line 3666 of app.jsx:
```
const TP_MARKER = "YOUR_TP_MARKER"
```

The `buildFlightUrl()` function (line 3685) checks if the marker equals this placeholder and skips it if so. This means **every flight click currently earns $0 commission.** The flight CTA is the most prominent monetization touchpoint in the app — it appears on ListingCard, CompactCard, and VenueDetailSheet.

**Fix:** Replace `YOUR_TP_MARKER` with the actual Travelpayouts marker from tp.media dashboard. This is a one-line change worth +$0.14 RPM immediately, and more importantly, it activates commission tracking on the highest-intent user action (booking flights).

**Jack action required:** Log into tp.media, grab the marker string, and either update line 3666 directly or share it so an agent can patch it.
